# ATS Resume Screening System

An AI-powered resume screening application that matches resumes with job descriptions
and provides ATS-style scores, skill gaps, and insights.

## Features
- Resume vs JD match score
- Skill extraction & categorization
- NLP-based semantic matching
- Recruiter-friendly interface

## Tech Stack
- Python
- Flask / Streamlit
- NLP (TF-IDF)
- Pandas, NumPy
